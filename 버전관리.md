## CLI 할 때 주의사항

1. 어디서 명령하고 있는지를 본다.
2. 명령하고 그게 제대로 됐는지 확인하는 방법까지 같이 익힌다.
3. 오류를 확인한다.

## CLI 명령어

- pwd : 현재 디렉토리 출력
- cd : 디렉토리 이동
  - .  : 현재 디렉토리
  - .. : 상위 디렉토리
- ls : 목록
- mkdir : 디렉토리 생성
- touch : 파일의 날짜와 시간을 수정
