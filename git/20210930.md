> `add` 와 `commit` 을 설명할 수 있는가?

* `add` 는 working directory 에서 staging area 로 넘기는 작업

  - ```bash
    $ git add . 	#  . 은 현재 위치 전체를 뜻함, 혹은 파일이나 폴더만도 가능
    ```

* `commit` 은 staging area 에서 커밋을 하는 작업

  - ```bash
    $ git commit -m '커밋메시지'	# '커밋메시지'로 커밋을 진행
    ```
